Excel导入数据工具类

功能作用:编写XML模板,读取Excel文件,然后入库。使用的Poi库,支持以xls,xlsx两种文件格式的导入。 其中一个导入类型的excel数据对应一个XML文件中的一个模板,模板之间根据id区分。 
代码思路:第一步,读取XML模板文件,根据id找到要导入excel文件数据对应的模板对象.
        第二步,读excel文件,返回list集合.
        第三步,根据第一步与第二步的结果,编写相应的sql语句并执行入库。
        注意事项:入库的过程中如果有转码或者类型校验的要求,代码在Service层,本例中转码使用Redis缓存。
